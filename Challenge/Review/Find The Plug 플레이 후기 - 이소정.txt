Find the plug : 도전에 도전하라 0.2ver


총평 : 1인이 3일만에 만들었다는 게 믿어지지 않음.
기본적인 기획 소양도 가지고 계신 것 같고, 재미있게 플레이했습니다!

물론 개발자님께서 더 개선하고 싶었지만 못 넣은 요소들도 분명 많을 것이라 생각합니다.
그래두 기획자+플레이어 입장에서 후기 약간 작성해 보겠습니다 ㅋㅋ


- 좋았던 점

1. 소재가 재미있음. 스토리를 게임잼 상황과 이어지게 만들어서 처음 컷씬 볼 때 몰입이 되었고, 병맛 컨셉도 더해져 흥미로웠다.
2. 자유도가 있음. 맵이 굉장히 넓어서 어떤 곳에서 배터리를 충전할 수 있는지 탐험한 후에야 알 수 있음.
배터리를 모두 충전한 후에도 집에 몰래 들어갈 수 있나 없나를 테스트해보려고 여러 집에 들락거렸다.
3. 문 앞에 가면 아래쪽에 조그맣게 설명이 나와서 도움말을 보고 키 조작을 할 수 있었다. 없었으면 헤맸을 듯.
4. 1234 버튼 누르면 tip UI 가 나오는 사실이 잘 인지됨. 하다가 궁금한 거 있으면 자주 이용할 수 있었음.
5. 가끔씩 예상치 못한 곳에서 콘센트를 발견했을 때 희열을 느낌
6. 더 개발하신다고 하면 추가로 넣을 수 있는 컨텐츠들이 굉장히 많음. 확장 가능성이 큰 게임.


- 수정되면 더 좋을법한것

1. 처음 시작 시 튜토리얼 스킵이 되지 않음. 컷씬처럼 스킵 가능했으면 좋겠다!
2. 게임 오버 시 다시 플레이하려고 했는데 그 상태에서 멈춤. 다시 플레이하려면 엔진을 다시 켜야 했음.
3. 집에 들어가지 않고 그냥 얻을 수 있는 콘센트가 많아서 (발견한 건 3개) 일단 걔네들부터 얻고 두번만 더 얻으면 쉽게 게임 클리어
그냥 얻을 수 있는 콘센트를 찾는 재미도 있기 때문에 없애는 것 보다는 다른 방법으로 해결하면 좋겠다.
처음에만 얻을 수 있게 하거나, 모아야 하는 배터리를 5개에서 7개 정도로 늘린다거나 등등.
4. 처음에 집에 침투하려고 문을 열었는데 '헬로' 해서 들킨 건 줄 알았음.
5. 힐팩이랑 은신 가능 아이템이 있다고 했는데 하나씩밖에 없는 건지?ㅜㅜ 좀 더 배치를 많이 해놔도 될 듯.


- 이 기능 추가되면 좋겟다!

1. npc 한테 말을 걸면 기본 대사가 나와서 게임 내 상황에 대해 설명하면 더 재밌을 것 같다.
ex) 	경찰1 : 이 곳에서 도난 사건이 일어나 수색 중입니다.
	경찰2 : 이 근방에서 수상한 사람을 목격하신다면 꼭 제보해 주십시오.
	도둑 : 빨리 이 마을을 떠야겠어. 더 있다간 체포될지도 몰라.
2. 문이나 창문을 똑똑 두드리면 집 안에 있는 npc 가 소리가 난 부분으로 이동하는 기능
3. 집 안의 npc 도 이동 혹은 시선 이동
